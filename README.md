## Install

```console
git clone https://github.com/dskyworks/ya-scooter.git
cd ya-scooter
yarn
```

## Usage

#### Dev

```console
yarn dev
```

#### Building

```console
yarn build
```

#### Preview

> Before preview, use yarn build

```console
yarn preview
```
